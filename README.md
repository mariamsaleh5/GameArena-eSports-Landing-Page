# GameArena eSports Landing Page UI

A responsive landing page UI for an eSports and gaming platform, designed to showcase games, team members, tournaments, and contact information.

##  Features
- Hero section with tagline, call-to-action buttons, and player stats
- Latest games showcase with categories (Adventure, Racing, Action)
- Services / What We Do section: Game Development, AR/VR, 3D Modeling
- Team section highlighting professional gamers, coaches, and content creators
- Contact section with form, location, and social links
- Fully responsive layout for desktop, tablet, and mobile
- Built with **Bootstrap 5** for modern UI components and responsiveness

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Responsive design principles

## Design Concept
This project simulates a **professional eSports platform**, focusing on:
- Clean, modern layouts
- Mobile-first responsiveness
- Interactive sections with visual hierarchy
- Engaging presentation of games and team members
